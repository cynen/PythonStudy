1. range()的使用:
    **range(stop) -> range object
    **range(start, stop[, step]) -> range object

    range(5) --> [0,1,2,3,4]  [0,5) -- 步进1
    range(0,5) --> [0,1,2,3,4]  [0,5) --默认步进1
    range(0,6,2) --> [0,2,4]  [0,6)  --步进2
    range(5,0,-1) --> [5,4,3,2,1] [5,0) -- 步进指定值.
